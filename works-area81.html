AREA81 HANDMADE SHOES Homepage site

Type : Web Design

Client : AREA81 HANDMADE SHOES 様

Web Site

  • Targetターゲット

    ブランドの価値観に興味を持った30代から60代の男女

  • Cnotext背景

    過去に立ち上げたWebサイトが機能しなくなったため、サイトのリニューアルを要望

  • Purpose目的

    商品のブランド価値を知ってもらう、興味を持ってくれた人に向けて信頼と安心を与える

  • Information情報設計

    ブランドの価値を知ってもらうために、ブランドイメージとなる言葉を強調したり、強みである“メンテナンスの手厚さ”について前サイトより詳しく記載することを意識しました。 安心感を得るために、新たに「ブランドヒストリー」の枠を設け、過去に行ってきた受注会等の実績が分かるようにしました。

  • Designデザイン

    配色はブランドイメージであるベージュとシンプルな無彩色で構成しました。 長文には可読性の高いフォントを使用し、安心感とわかりやすさに重点を置きました。 トップを横スクロールさせることで、横に“歩いている”ことをイメージさせ、インパクトを持たせました。また、ブランドのコンセプトである言葉、「履く人によって経年変化と共に美しく完成していく靴」という文字をファーストビューの次のページに配置しました。しっかり読んでもらうことを意識したため、言葉以外の情報量を少なくしました。その下にABOUTのリンクを配置して、興味を持った方にはすぐにページへ飛べるようにしています。
